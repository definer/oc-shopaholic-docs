### getValueString(_[$sSeparator = ', ']_)

Method returns string with property values.
```php
    $obList = PropertyValueCollection::make()->sort();
    
    echo $obList->getValueString();
    //result string: 'test1, test2, test10, test15'
    
    echo $obList->getValueString('-');
    //result string: 'test1-test2-test10-test15'
```
